# TransitScape
TransitScape: The Ultimate
Minecraft Transit Builder TransitScape is the definitive, highly-optimized modpack for creating and managing realistic public rail networks. Built around the powerful Minecraft Transit Railway (MTR), this pack includes essential addons like TJMetro, Transit Manager, and MSD to give you unparalleled control over routes, stations, and custom train models. Coupled with performance boosters (Sodium/Lithium) and crucial detailing tools (Chisels & Bits), TransitScape provides a complete, smooth, and immersive platform for designing everything from sprawling subway systems to high-speed intercity lines. Simply install and start connecting your world!

# TransitScape：终极 Minecraft 交通建设包
TransitScape 是一个权威性的、高度优化的模组包，专为创建和管理逼真的公共铁路网络而设计。该整合包以强大的 Minecraft Transit Railway (MTR) 为核心，包含 TJMetro、Transit Manager 和 MSD 等重要插件，让您对线路、车站和自定义列车模型拥有无与伦比的控制力。

结合了性能优化工具（Sodium/Lithium）和关键的细节美化工具（Chisels & Bits），TransitScape 提供了一个完整、流畅且沉浸式的平台，供您设计从庞大的地铁系统到高速城际线路的一切。

只需安装，即可开始连接您的世界！
